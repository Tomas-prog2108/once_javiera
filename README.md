# ¿Cómo crear un repositorio?

## Pasos

- Paso 1. Crear una cuenta de GitHub
- Paso 2. Entrar a GitHub
- Paso 3. Ir a la pestaña de repositorios
- Paso 4. Estando en repositorios, darle a un botón verde que está arriba a la derecha que dice **"New"**
- Paso 5. Ponerle nombre al repositorio y ***asegurarse de que sea público.***
- Paso 6. Darle a "Crear"
- Fin
